# Release Terms

ScratchBird public releases are source-review and technical-evaluation releases unless a release note says otherwise.

## Terms

`public source-review release` means the repository is available for build, inspection, test execution, and feedback.

`early beta` means the project is not production-ready and does not claim operational suitability, compatibility completeness, security certification, commercial support, or performance superiority.

`release-complete profile` means complete for the selected public source-review release profile. It is not a claim of production fitness.

`commercial readiness gate` means a gate that blocks known non-release artifacts from a release profile. It is not a claim that the public repository is commercially supported.

`supported driver` means a driver with current build, live route, conformance, packaging, and release evidence for the exact driver lane.

`tracked driver` means a driver or adaptor is inventoried but not released as supported.

`benchmark harness` means the project provides scripts or fixtures to run a workload. It is not a benchmark result.

`benchmark release validation` means captured run evidence with platform, compiler, route, workload, variance, logs, and hashes.

## Terms To Avoid In Public Claims

- production ready
- drop-in replacement
- cluster ready
- all drivers released
- AI production complete
- benchmarks prove faster
- commercial readiness proven

Use precise release evidence instead.
