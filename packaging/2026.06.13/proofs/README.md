# Public Proof Bundles

Only public-release proof artifacts belong here. Do not copy private workplans,
private audit notes, private reports, local paths, or private tracking material
into this directory.

Recommended proof families:

- `build/`
- `ctest/`
- `compatibility/`
- `security/`
- `sbom/`
- `signing/`
- `release-verifier/`
