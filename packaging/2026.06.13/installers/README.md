# Installers

Platform teams place finalized pre-release installer candidates under the
matching platform and target directory.

Expected layout:

- `windows-x64/msi/`
- `windows-x64/exe/`
- `linux-x86_64/deb/`
- `linux-x86_64/rpm/`
- `linux-x86_64/tar.gz/`
- `linux-arm64/deb/`
- `linux-arm64/rpm/`
- `linux-arm64/tar.gz/`
- `freebsd-x86_64/pkg/`
- `freebsd-x86_64/tar.gz/`
- `macos-arm64/pkg/`
- `macos-arm64/dmg/`
- `arch/aur-srcpkg/`
