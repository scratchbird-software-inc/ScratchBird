# Donor UDR Bridge Policy

Search key: `P6_DONOR_UDR_BRIDGE_POLICY_MANIFEST`

`DonorUdrBridgePolicyManifest.csv` is the P6 admission contract for donor parser-support and capability-reference UDR bridge behavior. It keeps ScratchBird engine MGA, security, storage, and recovery authority exclusive to the engine and permits donor UDR behavior only as parser support, metadata rendering, migration input, diagnostics, or exact refusal.
