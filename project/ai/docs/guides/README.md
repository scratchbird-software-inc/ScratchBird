# Guides

Practical, task-oriented documentation for running and validating ScratchBird AI.

- `GETTING_STARTED_EARLY_BETA.md` - bootstrap, local run, and live recertification flow for the current early-beta baseline.
- `INSTALL_UBUNTU_BETA1.md` - Ubuntu/Linux Beta 1 install and live bring-up path.
- `INSTALL_WINDOWS_BETA1.md` - Windows Beta 1 install and companion-host bring-up plus Windows recertification follow-up path.
- `RUNTIME_CONFIGURATION_AND_GOVERNED_OPERATION.md` - current runtime controls, durable approval behavior, quotas, resilience settings, and compatibility pins.
- `LIVE_BRIDGE_TROUBLESHOOTING.md` - direct-listener and local HTTP bridge troubleshooting for the current early-beta surface.
