## Action contract rule

Every action below requires evidence before success. Direct bounded actions may execute automatically only when registry authority class, policy gate, metric freshness, permission, and arbitration all allow the action. High-risk, destructive, security-sensitive, or cluster-scoped actions require explicit approval unless the action row states a bounded automatic path.
