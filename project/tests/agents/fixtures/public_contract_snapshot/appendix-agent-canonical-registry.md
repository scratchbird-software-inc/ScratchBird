## Registry row contract

| Field | Required content |
| --- | --- |
| `agent_type_id` | Stable identifier from the registry table. |
| `agent_uuid_rule` | Runtime instance UUIDv7 generated on first start for that scope and preserved across restart until agent instance is retired. |
| `scope` | Exact scope from registry row. |
| `runs_in` | Must be `database SBsvr/SBcore runtime` for built-in database agents or `cluster member SBsvr/SBcore/SBccore runtime` for agents supplied by admitted cluster libraries. Managers, listeners, parsers, drivers, and external tools are never agent hosts. |
| `default_install_state` | Installed state for new database, node, or cluster. |
| `default_runtime_state` | Enabled/disabled/recommend/observe/operator-only state at first boot. |
| `authority_class` | `observe_only`, `recommend_only`, `request_action`, `direct_bounded_action`, or `operator_only`. |
| `required_policies` | Exact policy family. |
| `required_metrics` | Exact metric families or named dependency group. |
| `optional_metrics` | Exact optional metric families. |
| `emitted_metrics` | Exact emitted agent metrics. |
| `allowed_actions` | Exact action IDs. |
| `forbidden_actions` | Explicit prohibited behavior. |
| `failure_mode` | Exact fail-closed behavior. |

## Examples

Success example: `memory_governor` starts on a database scope only after `memory_governor_policy` validates and required memory metrics are fresh. It enters `active` and emits `agent_transition_evidence`.

Denied example: `remote_query_routing_agent` cannot publish a route because route publication requires cluster fence authority. It may only create a routing recommendation.
