# Milvus donor catalog seed manifests

Search key: `SB_REFERENCE_DONOR_CATALOG_SEED_MILVUS`

This directory is reserved for exact newly-created Milvus donor catalog seed manifests.

Each supported donor version/profile must provide a manifest conforming to `SB_SPEC_DONOR_CATALOG_SEED_MANIFEST` before Milvus emulation setup can be considered implementation-ready.
