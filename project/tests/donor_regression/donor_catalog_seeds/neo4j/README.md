# Neo4j donor catalog seed manifests

Search key: `SB_REFERENCE_DONOR_CATALOG_SEED_NEO4J`

This directory is reserved for exact newly-created Neo4j donor catalog seed manifests.

Each supported donor version/profile must provide a manifest conforming to `SB_SPEC_DONOR_CATALOG_SEED_MANIFEST` before Neo4j emulation setup can be considered implementation-ready.
