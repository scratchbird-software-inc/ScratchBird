# MySQL and MariaDB donor catalog seed manifests

Search key: `SB_REFERENCE_DONOR_CATALOG_SEED_MYSQL_MARIADB`

This directory is reserved for exact newly-created MySQL and MariaDB donor catalog seed manifests.

Each supported donor version/profile must provide a manifest conforming to `SB_SPEC_DONOR_CATALOG_SEED_MANIFEST` before MySQL and MariaDB emulation setup can be considered implementation-ready.
