# InfluxDB donor catalog seed manifests

Search key: `SB_REFERENCE_DONOR_CATALOG_SEED_INFLUXDB`

This directory is reserved for exact newly-created InfluxDB donor catalog seed manifests.

Each supported donor version/profile must provide a manifest conforming to `SB_SPEC_DONOR_CATALOG_SEED_MANIFEST` before InfluxDB emulation setup can be considered implementation-ready.
