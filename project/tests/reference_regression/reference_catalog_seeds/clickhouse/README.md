# ClickHouse reference catalog seed manifests

Search key: `SB_REFERENCE_REFERENCE_CATALOG_SEED_CLICKHOUSE`

This directory is reserved for exact newly-created ClickHouse reference catalog seed manifests.

Each supported reference version/profile must provide a manifest conforming to `SB_SPEC_REFERENCE_CATALOG_SEED_MANIFEST` before ClickHouse emulation setup can be considered implementation-ready.
