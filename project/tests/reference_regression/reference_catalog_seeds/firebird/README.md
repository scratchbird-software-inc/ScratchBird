# Firebird reference catalog seed manifests

Search key: `SB_REFERENCE_REFERENCE_CATALOG_SEED_FIREBIRD`

This directory is reserved for exact newly-created Firebird reference catalog seed manifests.

Each supported reference version/profile must provide a manifest conforming to `SB_SPEC_REFERENCE_CATALOG_SEED_MANIFEST` before Firebird emulation setup can be considered implementation-ready.
