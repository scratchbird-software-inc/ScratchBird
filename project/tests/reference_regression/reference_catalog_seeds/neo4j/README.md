# Neo4j reference catalog seed manifests

Search key: `SB_REFERENCE_REFERENCE_CATALOG_SEED_NEO4J`

This directory is reserved for exact newly-created Neo4j reference catalog seed manifests.

Each supported reference version/profile must provide a manifest conforming to `SB_SPEC_REFERENCE_CATALOG_SEED_MANIFEST` before Neo4j emulation setup can be considered implementation-ready.
