# DuckDB reference catalog seed manifests

Search key: `SB_REFERENCE_REFERENCE_CATALOG_SEED_DUCKDB`

This directory is reserved for exact newly-created DuckDB reference catalog seed manifests.

Each supported reference version/profile must provide a manifest conforming to `SB_SPEC_REFERENCE_CATALOG_SEED_MANIFEST` before DuckDB emulation setup can be considered implementation-ready.
