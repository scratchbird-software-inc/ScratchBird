This directory is reserved for exact newly-created MySQL and MariaDB reference catalog seed manifests.

Each supported reference version/profile must provide a manifest conforming to `SB_SPEC_REFERENCE_CATALOG_SEED_MANIFEST` before MySQL and MariaDB emulation setup can be considered implementation-ready.
