# Milvus reference catalog seed manifests

Search key: `SB_REFERENCE_REFERENCE_CATALOG_SEED_MILVUS`

This directory is reserved for exact newly-created Milvus reference catalog seed manifests.

Each supported reference version/profile must provide a manifest conforming to `SB_SPEC_REFERENCE_CATALOG_SEED_MANIFEST` before Milvus emulation setup can be considered implementation-ready.
