# SBLR Surface Guardrail Tests

Search key: `SBLR-SURFACE-PUBLIC-GUARDRAIL-TESTS`

This directory contains project-owned guardrail and inventory tests for the
donor/SBLR interface closure substrate. The gates validate promoted fixture
counts, traceability shape, deterministic checked-in resources, and SBSQL sync
requirements. They do not implement storage, MGA, parser, listener, or server
runtime behavior.
